# Gallery Cache - 产品需求文档 (PRD)

## 1. 产品概述

**产品名称:** Gallery Cache
**产品类型:** 桌面端本地图片管理工具
**核心价值:** 安全、快速、高效地管理本地图片收藏，支持分级标签分类和瀑布流展示
**目标用户:** 设计师、摄影师、内容创作者等需要整理和管理大量图片的用户

## 2. 技术栈

| 层级 | 技术选型 |
|------|---------|
| 后端框架 | Tauri 2.x (Rust) |
| 前端框架 | React 18 + TypeScript |
| 状态管理 | Zustand |
| 瀑布流布局 | react-masonry-css |
| 样式 | Tailwind CSS v4 |
| 构建工具 | Vite |
| 图片处理 | image crate (Rust) |

**跨平台支持:** Windows 10/11 (x64)、macOS 11+ (Intel & Apple Silicon)

## 3. 核心功能

### 3.1 文件夹管理

| 功能 | 描述 |
|------|------|
| 首次启动引导 | 无配置时显示欢迎界面，引导用户选择存储目录 |
| 目录选择 | 使用系统对话框选择本地文件夹作为数据存储位置 |
| 配置持久化 | 将路径保存到 `~/.gallery-cache/config.json` |
| 数据目录结构 | `{dataPath}/images/` + `{dataPath}/metadata/` |

### 3.2 标签管理

| 功能 | 描述 |
|------|------|
| 创建标签 | 支持一级标签和子标签（二级、三级） |
| 编辑标签 | 双击或右键重命名 |
| 删除标签 | 右键删除，图片变为未分类状态 |
| 标签排序 | 同级标签按 order 字段排序 |
| 拖拽分配 | 可将图片拖拽到标签上分配关系 |

### 3.3 图片展示

| 功能 | 描述 |
|------|------|
| 瀑布流布局 | 自适应列数（最小 200px/列） |
| 网格布局 | 固定宽高比（1:1）网格展示 |
| 视图切换 | 瀑布流 ↔ 网格 切换 |
| 排序方式 | 按日期 / 按名称 / 按大小 |
| 搜索过滤 | 按文件名搜索图片 |

### 3.4 图片导入

| 方式 | 描述 |
|------|------|
| 粘贴导入 | Ctrl/Cmd + V 从剪贴板粘贴图片 |
| CDN 导入 | 输入 URL 批量下载图片 |
| 拖拽导入 | 从文件管理器拖拽图片到应用内 |

### 3.5 图片查看器

| 功能 | 描述 |
|------|------|
| 全屏预览 | 点击图片打开全屏查看器 |
| 缩放 | 滚轮缩放（50%-300%）|
| 平移 | 放大后可拖拽平移 |
| 导航 | 键盘左右箭头 / 点击箭头切换图片 |
| 快捷键 | ESC 关闭、+/- 缩放、0 重置 |

### 3.6 图片操作

| 功能 | 描述 |
|------|------|
| 右键菜单 | 查看详情、添加标签、复制图片、删除 |
| 标签分配 | 点击"标签"按钮选择标签 |
| 删除确认 | 删除前弹出确认对话框 |
| 复制图片 | 复制图片到剪贴板 |

## 5. 数据模型

### 5.1 配置文件

```json
{
  "version": 1,
  "dataPath": "~/gallery-cache-data",
  "theme": "dark",
  "lastOpenedAt": 1704067200000
}
```

### 5.2 标签数据 (tags.json)

```json
{
  "tags": [
    { "id": "uuid", "name": "抽象分类", "parentId": null, "order": 0, "createdAt": 1704067200000 }
  ]
}
```

### 5.3 图片元数据 (images.json)

```json
{
  "images": [
    {
      "id": "uuid",
      "filename": "uuid.jpg",
      "originalName": "photo.jpg",
      "width": 1920,
      "height": 1080,
      "size": 204800,
      "createdAt": 1704067200000,
      "importedAt": 1704067200000,
      "source": "paste" | "cdn" | "file"
    }
  ]
}
```

### 5.4 图片-标签关系 (image-tags.json)

```json
{
  "relations": [
    { "imageId": "img-uuid", "tagId": "tag-uuid" }
  ]
}
```

## 6. 验收标准

### 6.1 功能验收

| 功能 | 验收条件 |
|------|---------|
| 文件夹选择 | 首次启动弹出文件夹选择，重启后记住路径 |
| 标签增删改 | 可创建多级标签，可重命名，可删除 |
| 瀑布流展示 | 图片正确排列，滚动流畅 |
| 图片粘贴 | Ctrl/Cmd + V 可从剪贴板导入图片 |
| CDN 导入 | 输入 URL 可下载图片到本地 |
| 图片查看 | 点击图片可全屏查看，支持缩放和导航 |
| 数据持久 | 重启后图片、标签、关系完整保留 |

### 6.2 性能要求

| 指标 | 要求 |
|------|------|
| 启动时间 | < 3 秒 |
| 图片加载 | 首屏 < 1 秒（100 图） |
| 滚动帧率 | >= 50 FPS |

### 6.3 视觉验收

- 暗色主题一致
- 颜色、字体、间距符合规格
- 悬停、选中状态可见
- 响应式布局正确
