# Gallery Cache - 本地图片管理工具

## 1. 项目概述

**项目名称:** Gallery Cache
**类型:** 桌面端本地图片管理工具
**核心功能:** 两栏布局管理本地图片，支持分级标签分类和瀑布流展示
**目标用户:** 设计师、摄影师、内容创作者等需要整理和管理大量图片的用户

### 技术栈

- **后端框架:** Tauri 2.x (Rust)
- **前端框架:** React 18 + TypeScript
- **状态管理:** Zustand
- **瀑布流:** react-window + masonry 虚拟化
- **样式:** Tailwind CSS
- **构建:** Vite

### 跨平台支持

- Windows 10/11 (x64)
- macOS 11+ (Intel & Apple Silicon)

## 3. 功能规格

### 3.1 文件夹管理

#### 初始化流程

1. 应用启动
2. 检查本地配置 (`~/.gallery-cache/config.json`)
3. 若无配置或路径无效 → 显示欢迎界面，引导用户选择文件夹
4. 若有配置 → 加载数据目录

#### 文件夹选择

- 使用 Tauri dialog API (`open` with `directory: true`)
- 用户选择后，保存到 `~/.gallery-cache/config.json`
- 数据目录结构:
  ```
  ~/gallery-cache-data/
  ├── images/
  │   ├── {uuid}.jpg
  │   ├── {uuid}.png
  │   └── ...
  └── metadata/
      ├── tags.json
      └── image-tags.json
  ```

### 3.2 标签管理

#### 数据模型

```typescript
interface Tag {
  id: string;           // UUID
  name: string;         // 标签名
  parentId: string | null; // 父标签 ID (null = 一级标签)
  order: number;        // 排序序号
  createdAt: number;    // 创建时间戳
}

interface ImageTagRelation {
  imageId: string;
  tagId: string;
}
```

#### 操作

| 操作 | 触发方式 | 行为 |
|------|----------|------|
| 添加一级标签 | 点击 "+" 按钮 | 弹出输入框，创建后添加到列表顶部 |
| 添加子标签 | 右键菜单 / 选中父标签后点击 "+" | 在父标签下创建 |
| 重命名 | 右键菜单 / 双击标签名 | inline 编辑 |
| 删除 | 右键菜单 | 确认弹窗，删除后图片变为"未分类" |
| 拖拽排序 | 拖拽 | 同级内调整顺序 |

### 3.3 图片展示

#### 瀑布流布局

- 使用虚拟化列表优化 (react-window)
- 列数根据容器宽度自适应 (min 200px)
- 懒加载: 只渲染可视区域 ± 缓冲区的图片
- 缩略图: 最大边 400px，质量 80%

#### 图片操作

| 操作 | 行为 |
|------|------|
| 点击图片 | 打开图片查看器 (全屏 + 导航) |
| 右键菜单 | 查看详情、复制、删除 |
| 拖拽图片 | 可拖入标签区域 |

### 3.4 图片导入

#### 粘贴导入

- 监听全局 `paste` 事件 (Command/Ctrl + V)
- 获取剪贴板中的图片数据
- 保存到 `images/` 目录
- 弹出标签选择框，用户选择目标标签
- 支持多图连续粘贴

#### CDN URL 导入

- 工具栏 "添加" 按钮 → "从 URL 导入"
- 输入框支持单个或批量 URL (换行分隔)
- 显示下载进度条
- 下载完成后自动保存到 `images/` 目录
- 解析规则:
  - 从 URL 提取文件名，无则用 UUID
  - 支持 `?` 参数自动去除

#### 拖拽导入

- 支持从系统文件管理器拖拽图片到瀑布流区域
- 支持从其他应用拖拽图片

### 3.5 图片查看器

- 全屏模态框
- 居中显示原图 (保持比例)
- 左右箭头导航 (键盘 + 点击)
- ESC 或点击遮罩关闭
- 显示图片名、尺寸、文件大小

## 4. 数据持久化

### 配置文件 (`~/.gallery-cache/config.json`)

```json
{
  "version": 1,
  "dataPath": "~/gallery-cache-data",
  "theme": "dark",
  "lastOpenedAt": 1704067200000
}
```

### 标签数据 (`{dataPath}/metadata/tags.json`)

```json
{
  "tags": [
    { "id": "uuid-1", "name": "抽象分类", "parentId": null, "order": 0, "createdAt": 1704067200000 },
    { "id": "uuid-2", "name": "灵感收集", "parentId": "uuid-1", "order": 0, "createdAt": 1704067200001 }
  ]
}
```

### 图片-标签关系 (`{dataPath}/metadata/image-tags.json`)

```json
{
  "relations": [
    { "imageId": "img-uuid-1", "tagId": "uuid-2" }
  ]
}
```

### 图片元数据 (`{dataPath}/metadata/images.json`)

```json
{
  "images": [
    {
      "id": "img-uuid-1",
      "filename": "uuid-1.jpg",
      "originalName": "photo.jpg",
      "width": 1920,
      "height": 1080,
      "size": 204800,
      "createdAt": 1704067200000,
      "importedAt": 1704067200000,
      "source": "paste" | "cdn" | "file"
    }
  ]
}
```

## 5. 验收标准

### 功能验收

| 功能 | 验收条件 |
|------|----------|
| 文件夹选择 | 首次启动弹出文件夹选择，重启后记住路径 |
| 标签增删改 | 可创建多级标签，可重命名，可删除 |
| 瀑布流展示 | 图片正确排列，滚动流畅 (1000+ 图) |
| 图片粘贴 | Ctrl/Cmd + V 可从剪贴板导入图片 |
| CDN 导入 | 输入 URL 可下载图片到本地 |
| 图片查看 | 点击图片可全屏查看，可导航 |
| 数据持久 | 重启后图片、标签、关系完整保留 |

### 性能验收

- 启动时间: < 3 秒
- 图片加载: 首屏 < 1 秒 (100 图)
- 滚动帧率: >= 50 FPS (1000+ 图)
- 内存占用: < 500MB (1000 图)

### 视觉验收

- 颜色、字体、间距符合 SPEC
- 暗色主题一致
- 响应式布局正确
- 悬停、选中状态可见
